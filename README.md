
# techNews

## Опис проєкту
Веб-застосунок для автоматичного збору технічних новин з популярних RSS-стрічок TechCrunch, DOU. Проєкт дозволяє переглядати новини в єдиному інтерфейсі, має панель адміністратора для управління контентом та базу даних для архівування новин.

Проєкт виконано в рамках навчальної практики Тижні 1-4.

## Технологічний стек
    Backend: Python 3.10 + Flask
    Database: PostgreSQL 15
    Frontend: HTML5, CSS3, Bootstrap 5
    Containerization: Docker, Docker Compose
    Security: Werkzeug (password hashing), Flask-Session

## Функціонал
1.  Автоматичний парсинг: Фоновий процес збирає новини з RSS-джерел щогодини.
2.  Стрічка новин: Відображення останніх новин з сортуванням за датою.
3.  Адміністративна панель:
    Реєстрація та авторизація адміністраторів.
    Ручне додавання новин.
    Видалення новин.
4.  Безпека: Захист паролів (хешування), захист сесій, захист від SQL-ін'єкцій (ORM/psycopg2 parameters).
